🔗 Firebase 동기화 설정

Firebase Realtime Database를 연결하면 PC와 핸드폰이 실시간으로 동기화됩니다.
아직 Firebase 프로젝트가 없다면 console.firebase.google.com에서 무료로 만드세요.

설정 방법
1. Firebase Console → 프로젝트 생성
2. 빌드 → Authentication → Sign-in method에서 익명(Anonymous) 로그인 활성화
3. 빌드 → Realtime Database 메뉴에서 DB 생성
4. 규칙(Rules) 탭에서 아래 규칙으로 변경 후 게시 (인증된 사용자만 읽기/쓰기 가능):
{ "rules": { ".read": "auth != null", ".write": "auth != null" } }
5. 데이터(Data) 탭 상단 URL 복사 (예: https://xxx.firebaseio.com)
6. 앱이 없다면 프로젝트 설정(톱니바퀴) → 일반 → 내 앱에서 </> (웹) 아이콘을 눌러 웹 앱을 먼저 등록
7. 등록 시 나오는 firebaseConfig 코드에서 apiKey 값을 복사 (appId 아님 — 아래 참고)
8. 아래 필드에 URL과 API Key를 붙여넣고 저장
⚠️ 익명 로그인을 활성화하지 않으면 동기화가 실패합니다.

apiKey vs appId 헷갈릴 때:
apiKey: "AIzaSy..." ← 이 값을 사용 (Web API Key)
appId: "1:123...:web:abc" ← 이건 사용 안 함
