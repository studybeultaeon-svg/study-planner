🔗 Firebase 동기화 설정

Firebase Realtime Database를 연결하면 PC와 핸드폰이 실시간으로 동기화됩니다.
아직 Firebase 프로젝트가 없다면 console.firebase.google.com에서 무료로 만드세요.

설정 방법
1. Firebase Console → 프로젝트 생성
2. 빌드 → Realtime Database 메뉴에서 DB 생성
3. 규칙(Rules) 탭에서 아래 규칙으로 변경 후 게시:
{ "rules": { ".read": true, ".write": true } }
4. 데이터(Data) 탭 상단 URL 복사 (예: https://xxx.firebaseio.com)
5. 아래 필드에 URL을 붙여넣고 저장
